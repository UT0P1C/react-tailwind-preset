## Overview

### Funções

Usuários podem:

- Criar projetos react
- Adicionar Typescript
- Adicionar TailwindCSS

### Como utilizar

```
$ npm i -g tailwind-react-preset
```

Após a instalação, no diretório onde quer criar o projeto:

```
$ tailwind-react-preset
```
